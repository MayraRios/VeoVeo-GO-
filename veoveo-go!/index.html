VEO VEO GO!

version 1.0

Why?

While taking Computer Science & Javascript certification, I came up with the seed idea of this game. The business model behind this prototype is to create a template in such a way that merchants can acquire a customized version and thus create an on-site experience in their business.

How is the game?

The game has two parts:
A) First part: Given the first letter and given how many letters it has, you have to guess a word. After 3 chances you will know the word that is the name of the object to search.
B) Second part: You have 20 seconds to search for the object. Place the link on the movil phone and look for the object between the markers. The libraries and resources used in this software makes that only the movil phones with Android operating system are able to run augmented reality framework. Sorry for that! Today there is a solution for this problem with Tango platform by Google. This will be available in the next version.

What is the game about?

I decided to bring back the silly old games to today’s digital age to maintain the bond between old and future generations. In addition, I took advantage of the moment and introduced images and informational capsules of COVID-19 to keep children informed.
